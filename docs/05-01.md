## 提问

```text
请模仿金庸武侠的文笔风格，写一篇 1200 字左右的原创短篇武侠小说。
要求：半文半白、江湖气息浓厚、有人物名号、简易武功对决、带一点家国侠义感、结局留白有韵味，不要用金庸原著人物，全部原创角色和江湖门派。
```

## 短视频

1. **AI 配音**
用剪映配音、讯飞配音，选儒雅古风男声，完美贴合武侠感。
2. **配画面**
剪映搜：古风江湖、武侠山水、侠客独行、竹林、客栈、刀剑打斗 素材，直接套用模板。
3. **加字幕 + BGM**
配古风纯音乐（古筝、箫、江湖配乐），大字古风字幕，一条视频就做好了。

## 避坑重点

1. 不要直接用金庸原著人物名（郭靖、杨过、令狐冲这些别用），全部原创人名门派，规避版权。
2. 不要大段复制原著原文，只模仿文笔、原创剧情。
3. 文案控制在800–1500 字，对应视频 1–3 分钟，最适合短视频完播率。

```text
残雪覆岸，寒江呜咽。
渡口酒旗猎猎，上书“望归楼”三字，被朔风扯得噼啪作响。楼中只坐一人，青衫洗得发白，腰间悬一柄乌木剑鞘的长剑，剑穗是褪色的蓝绫，随他抬杯的动作轻轻晃动。
此人姓苏，名砚秋，江湖人送外号“寒江一剑”。十年前，他是江南苏家最得意的子弟，苏家以“流云剑法”名震武林，却因不愿依附东厂，一夜之间被灭门。苏砚秋彼时年幼，被忠仆拼死救出，苦练剑法十年，只为寻得当年带头灭门的东厂千户——魏屠。
忽闻马蹄声骤响，尘土飞扬，十数名黑衣劲装之人簇拥着一个满脸横肉的大汉，踏雪而来。大汉腰佩弯刀，脸上一道刀疤从眉骨延伸至下颌，正是魏屠。他今日奉东厂之命，护送一批赃银前往京城，途经此渡口，便要进店歇息。
店小二连忙上前躬身伺候，魏屠一脚踹开，厉声喝道：“瞎眼的东西！快上最好的酒肉，耽误了公公的事，拆了你这破楼！”
苏砚秋缓缓放下酒杯，抬眼望去，目光如寒江冰棱，直刺魏屠。“魏千户，十年不见，倒是越发威风了。”
魏屠闻言一怔，随即打量起苏砚秋，半晌才狞笑道：“原来是苏家的漏网之鱼！当年没斩草除根，倒是让你苟活至今。今日既然撞上，正好了了当年的事！”说罢，挥手示意手下动手。
两名黑衣人身如鬼魅，拔刀直扑苏砚秋，刀风凌厉，带着刺骨的寒意。苏砚秋身形微动，腰间长剑“呛啷”出鞘，剑光如流云般闪过，只听“当啷”两声脆响，两名黑衣人的弯刀竟被一剑挑飞，剑势未歇，剑尖已抵在二人咽喉。
“好快的剑！”魏屠眼中闪过一丝忌惮，随即怒吼一声，亲自提刀上前。他的刀法刚猛霸道，每一刀都带着东厂制式武学的狠辣，刀刀直取要害。苏砚秋不慌不忙，流云剑法施展开来，剑影飘忽，如流水绕山，看似轻柔，却招招暗藏杀机。
寒江之上，风雪更急。楼外刀剑相撞之声不绝于耳，火星在白雪中飞溅。魏屠的刀法越来越乱，额上青筋暴起，他没想到十年时间，苏砚秋的剑法竟已臻至如此境界。
“噗——”苏砚秋抓住一个破绽，长剑斜挑，剑尖划过魏屠的手腕，弯刀“当啷”落地。他顺势上前，长剑直指魏屠心口，“当年苏家满门鲜血，今日，该你偿命了。”
魏屠面如死灰，却仍色厉内荏：“你敢杀我？东厂不会放过你的！”
苏砚秋眼中寒光一闪，长剑微微一送，魏屠闷哼一声，倒在雪地里，没了气息。其余黑衣人大惊失色，四散奔逃，却被苏砚秋身形一晃，剑影翻飞，尽数倒地。
苏砚秋收剑入鞘，看着满地尸体，脸上没有丝毫波澜，只有眼底深处藏着一丝难以察觉的疲惫。他拿起桌上的酒壶，一饮而尽，将酒壶往地上一掷，转身踏雪走向寒江。
风雪中，他的青衫身影越来越远，最终消失在江雾之中。只留下望归楼前的残雪，和那柄悬在腰间的长剑，在寒风中轻轻作响。
江湖路远，侠义难寻。寒江孤剑，不知又要奔赴何方。
```
